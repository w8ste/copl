1. What is the difference between first-order functions and higher-order functions? Give an example for a first-order function and for a higher-order function!
   #+begin_src python
     '''
     First-order functions are functions that neither return nor take other functions are
     parameters. They parameterise computations over data.
     Higher-order functions on the other hand return and/or take other functions as parameters
     and parameterize computations over other computations.
     '''
     # First order
     def add(n) -> Int:
         return n + 3

     # Higher order
     def apply_operation(x, operation):
         return operation(x)
   #+end_src

   #+begin_src python
     '''
     It means that functions are values with all the rights of other values. As long as the languages support functions, which accept parameters and return values, then first-class implies higher-order functions.
     '''
